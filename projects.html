Verbo Vista

Verbo Vista is a comprehensive vocabulary learning application designed to enhance users' language skills through interactive quizzes and personalized learning paths. The app provides a dynamic and engaging platform for users to expand their vocabulary effectively.

  • Key Features:
  • Interactive Quizzes: Users can participate in various quizzes to test and improve their vocabulary.
  • Personalized Learning Paths: Tailored learning experiences based on user progress and proficiency.
  • Progress Tracking: Visualize and track your learning journey with detailed progress reports.
  • User-Friendly Interface: Designed with an intuitive interface to make learning enjoyable and accessible.

Real-Time Data Visualization Dashboard

Developed a full-stack web application designed for real-time data visualization and interactive user engagement. The application features a React frontend and an Express/Node.js backend, leveraging WebSocket technology to provide live data updates.

  • Key Features:
  • Real-Time Data Updates: Implemented WebSocket communication between the server and client to push live data updates every 5 seconds, ensuring users receive the most current information.
  • Interactive Visualization: Integrated D3.js for dynamic and interactive visualizations, allowing users to explore and interact with data in real-time.
  • User Authentication: Implemented secure user authentication and session management using JWT (JSON Web Tokens) to protect user data and access.
  • Responsive Design: Designed a responsive user interface using React and CSS, ensuring usability across various devices and screen sizes.
  • Deployment: Deployed the application using Heroku and AWS, providing scalable and reliable access to users globally.
